# Чек-лист проверки кандидата (Background Check) в РК

## Этап 1. Документальный анализ
- [ ] Проверить действительность Удостоверения личности через сервис проверки документов КГД/Egov.
- [ ] Сверить соответствие ИИН дате рождения и полу.

## Этап 2. Проверка рисков благонадежности (По ИИН)
- [ ] Поиск в реестре должников АИС ОИП (наличие крупных задолженностей, ограничений).
- [ ] Поиск по Судебному кабинету (уголовные дела, экономические споры, иски).
- [ ] Скрининг списка лиц, связанных с финансированием деструктивных организаций (АФМ).

## Этап 3. Анализ цифрового следа (OSINT)
- [ ] Поиск профилей в социальных сетях (LinkedIn, Facebook).
- [ ] Проверка телефонного номера через агрегаторы тегов для выявления скрытых связей.
